    Material for MkDocs*

    • Material for MkDocs

    material 是 mkdocs 的一个主题，提供多种配置选项，这里记录如何开启并利用一些特性。

    主要是主题配置和插件支持。

    Markdown Extensions*

    mkdocs 基于 Python Markdown 解析 markdown，后者为标准 markdown 提供了扩展。

    pymdown-extensions 为 Python Markdown 提供更多的扩展功能。

    都要在 mkdocs.yml 中开启。

    markdown_extensions:
  - admonition
  - md_in_html
  - codehilite:
      linenums: true
      guess_lang: false
  - def_list
  - footnotes
  - meta
  - toc:
      permalink: "*"
  - pymdownx.arithmatex
  - pymdownx.caret
  - pymdownx.critic
  - pymdownx.details
  - pymdownx.emoji:
      emoji_index: !!python/name:materialx.emoji.twemoji ""
      emoji_generator: !!python/name:materialx.emoji.to_svg ""
  - pymdownx.highlight      
  - pymdownx.inlinehilite
  - pymdownx.keys
  - pymdownx.magiclink
  - pymdownx.mark
  - pymdownx.smartsymbols
  - pymdownx.superfences:
      custom_fences:
        - name: mermaid
          class: mermaid
          format: !!python/name:pymdownx.superfences.fence_code_format ""  
  - pymdownx.betterem:
      smart_enable: all
  - pymdownx.tasklist:
      custom_checkbox: true
  - pymdownx.tilde

    Admonitions 警告*

    其实就是文档的提示块，开启支持：

    markdown_extensions:
  - admonition
  - pymdownx.details
  - pymdownx.superfences

    !!! note "Title"

    Title

    Code blocks 代码块*

    包括语法高亮、嵌套代码块、snippets 等功能，开启支持：

    markdown_extensions:
  - pymdownx.highlight
  - pymdownx.inlinehilite
  - pymdownx.superfences
  - pymdownx.snippets

    支持给代码块添加标题、添加行内注释（8.0.0b1 的特性，还不能用）、行号（甚至支持不从 1 开始）、高亮选中行。

    markdown 语法：

    ```yaml title="yaml" linenums="2" hl_lines="1 2"
theme:
  features:
    - content.code.annotate # (1)
```

  features:
    - content.code.annotate # (1)

    Content tabs 标签*

    开启支持：

    markdown_extensions:
  - pymdownx.superfences
  - pymdownx.tabbed:
      alternate_style: true

    支持组合代码块。

    markdown 语法：

    === "C"

    ```c
    #include<stdio.h>

    int main() {
      printf("Hello World!\n");
      return 0;
    }
    ```

=== "C++"

    ```c++
    #include<iostream>

    int main() {
      std::cout << "Hello World!" << std::endl;
      return 0;
    }
    ```

    渲染结果：

    #include<stdio.h>

int main() {
  printf("Hello World!\n");
  return 0;
}

    #include<iostream>

int main() {
  std::cout << "Hello World!" << std::endl;
  return 0;
}

    开启 pymdownx.superfence 之后，块之间随意嵌套。

    Formatting 格式化*

    开启支持：

    markdown_extensions:
  - pymdownx.critic
  - pymdownx.caret
  - pymdownx.keys
  - pymdownx.mark
  - pymdownx.tilde

    开启 pymdownx.critic 之后，支持高亮修改或者添加一段 inline 注释。

    markdown 语法：

    Text can be {`--`deleted`--`} and replacement text {`++`added`++`}. This can also be
combined into {`~~`one`~>`a single`~~`} operation. {`==`Highlighting`==`} is also
possible {`>>`and comments can be added inline`<<`}.

{`==`

Formatting can also be applied to blocks by putting the opening and closing
tags on separate lines and adding new lines between the tags and the content.

`==`}

    渲染结果：

    Text can be deleted and replacement text added. This can also be combined into onea single operation. Highlighting is also possible and comments can be added inline.

    Formatting can also be applied to blocks by putting the opening and closing tags on separate lines and adding new lines between the tags and the content.

    Icons + Emojis 图标和表情*

      emoji_generator: !!python/name:materialx.emoji.to_svg

      支持图标和标签，集成了 Material Design, FontAwesome, Octicons 图标库。

      markdown 语法：

      :smile:

:material-account-circle: - `material/account-circle.svg`

      渲染结果：

      - material/account-circle.svg

      Images 图像*

      开启支持：

      markdown_extensions:
  - attr_list
  - md_in_html

      支持图片靠左或右对齐。

      markdown 语法：

      ![](https://dummyimage.com/600x400/eee/aaa){ align=left width=300}

      渲染结果：

      这是一张图片。

      markdown 语法：

      ![](https://dummyimage.com/600x400/eee/aaa){ align=right width=300}

      渲染结果：

      这是一张图片。

      markdown 不支持图像标题，使用 html figure 和 figcaption 标签。

      markdown 语法：

      <figure markdown>
  ![Dummy image](https://dummyimage.com/600x400/){ width="300" }
  <figcaption>图片标题</figcaption>
</figure>

      渲染结果：

      使用 {loading=lazy} 可以设置图片延迟加载。

      markdown_extensions:

  - pymdownx.tasklist:

      definition

      description
